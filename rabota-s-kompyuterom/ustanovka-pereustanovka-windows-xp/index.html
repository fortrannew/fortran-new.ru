Установка Windows XP

Установка Windows XP вначале требует тщательной подготовки. Установить её сможет каждый. Главное - иметь все необходимое для этого:

  • Инсталляционный диск с операционной системой (ОС).
  • На обратной стороне коробки диска обязательно должен быть написан верный регистрационный код.
  • Исправный CD-рекордер. Это тоже важный момент, так как без него диск читаться не будет и дальнейшая установка окажется невозможной.
  • Съемный носитель информации, на который предварительно скопирована вся ценная информация с компьютера.

Итак, помещаем инсталляционный диск в CD-привод и перезагружаем компьютер. Как только компьютер станет загружаться, нажимаем и удерживаем клавишу "Delete", пока не появится "Bios". Теперь, используя кнопки со стрелками на клавиатуре ("Вверх", "Вниз", "Вправо", "Влево"), находим пункт "First Boot Device" (или "1st Boot Device") нажимаем "Enter" и выбираем пункт "CD-ROM". Структура разных версий "Bios" на каждом компьютере разная, поэтому и название того раздела, в котором находится "First Boot Device", отличается. Однако просматривая поочередно все разделы "Bios", этот пункт найти не сложно. Значения других параметров "Bios" менять не нужно, если не уверены в правильности своих действий.

После того, как указали для "First Boot Device" значение "CD-ROM", нажимаем кнопку "F10" и затем "Enter". Если всё выполнено верно, то буквально через несколько секунд компьютер попросит подтвердить загрузку с компакт-диска надписью "Press any key to boot from CD". Очень важно именно в этот момент успеть нажать любую клавишу, иначе придется опять перезагружать компьютер и опять успевать нажимать любую клавишу.

Затем компьютер начнет установку "Windows" (появится синий экран с надписью "установка Windows"). Пользователю ничего делать не нужно, пока сам компьютер об этом не "попросит".

Первый вопрос компьютера содержит выбор между тем, чтобы продолжить установку, восстановить старую систему Windows XP (в случае, когда производится переустановка ОС) или же отказаться от дальнейшей установки. Для продолжения нажимаем "Enter".

Потом принимаем лицензионное соглашение и жмем F8.

Дальше отказываемся от восстановления старой ОС (при переустановки) и жмем "Esc". Появляется окно, где мы должны будем разделить емкость винчестера на логические элементы.

При переустановке ОС, выделяем раздел C, жмем кнопку "D", затем "Enter" и "L". Поздравляем! Старой операционной системы теперь больше нет. Она была безвозвратно удалена. Точно также, если нужно, удаляем и другие разделы. Но те разделы, на которых хранилась ценная информация, которая не была предварительно скопирована на съемные носители, ни в коем случае не удаляйте! Потом Вы её уже не вернете.

Дальше начинаем делить неразмеченную область винчестера на логические диски. Используя клавиши "Вверх" и "Вниз", выделяем "Неразмеченная область" и жмем кнопку "C". Появится серое окошко, где вводим объем будущего логического диска C в МБ и нажимаем "Enter".

В списке разделов появился новый диск, а неразмеченная область сократилась на его размер. Оставшуюся емкость таким же образом разделяем на один или несколько разделов (на Ваше усмотрение).

Затем перемещаем курсор на раздел С, куда будем устанавливать операционную систему, и жмем "Enter". Программа предложит 4 (2) варианта форматирования ОС. Выбираем "Форматировать раздел в системе NTFS" и нажимаем "Enter".

Дальше пользователю ничего делать не нужно, так как программа сама будет форматировать выбранный раздел и копировать на него все необходимые файлы с инсталляционного диска. Через некоторое время откроется окно "Язык и региональные стандарты". Нажимаем мышкой верхнюю кнопку "Настроить". На вкладке "Региональные параметры" в соответствующих полях выбираем свой язык и страну, где располагается компьютер, и нажимаем кнопку "OK".

Потом нажимаем нижнюю кнопку "Настроить". Выберите те языки, которые необходимы будут для работы с текстовой информацией. Для этого нажмите "Добавить", в списке "Язык ввода" выбираете нужный язык и нажимаете кнопку "OK". Удалить лишний язык можно, выделив его мышкой и нажав кнопку "Удалить". Затем в списке "Язык ввода по умолчанию" выбираем язык, который чаще всего будет использоваться на компьютере и нажимаем кнопку "Применить", затем "Ок" и "Далее", после чего вводим имя пользователя (обязательно, иначе процесс установки не завершится) и название организации (не обязательно), нажимаем кнопку "Далее".

Откроется окно "Ключ продукта", куда нужно будет ввести код устанавливаемой Windows XP (регистрационный код, который заранее должен быть подготовлен) и нажать "Далее". Чтобы сменить раскладку клавиатуры и печатать латинскими (русскими) буквами, нажмите "Shift",  удерживая кнопку "Alt".

Если необходимо, вводим в окно "Имя компьютера и пароль администратора" одинаковый пароль в поля "Пароль администратора" и "Подтверждение" (обязательно запомните этот пароль).

В следующем окне указываем страну и телефонный код города, в котором находится компьютер и жмем "Далее".

Потом настраиваем дату, время, выбираем часовой пояс, где находится компьютер, выбираем "Автоматический переход на летнее время и обратно", нажимаем "Далее".

Затем программа продолжит установку. А через некоторое время попросит ввести сетевые параметры. Отмечаем окошко "Обычные параметры", жмем "Далее". Потом ничего не меняя нажимаем "Далее". Любые изменения настроек сети при необходимости можно будет произвести после установки ОС. Ждем окончания установки Windows.

В конце установки произойдет перезагрузка и первый запуск Windows. Появятся два окна для оптимизации разрешения монитора. Нажимаем "OK", после чего откроется мастер первого запуска. Отвечайте на вопросы компьютера и нажимайте "Далее". Как только дойдете до раздела "Пользователи компьютера", перезагрузите компьютер, нажав "Reset" на системном блоке - учетные записи лучше настроить позже.

Во время перезагрузки заходим в "Bios" (удерживая кнопку "Delete", и меняем значение "First Boot Device" с "CD-ROM" на "HDD0", нажимаем F10, затем "Enter".

Как только компьютер запустится, извлекаем инсталляционный диск из привода. Установка операционной системы Windows XP успешно завершена.
